# Descripción

Programa para encontrar imágenes iguales o similares aunque se haya cambiado su peso, por ejemplo la imagen original y la recibida por WhatsApp.

Usa hashes perceptuales (phash, ahash, dhash, whash) para detectar duplicados incluso después de compresión o redimensión.
